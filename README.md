# Barbearia Victor Style

Um site institucional fictício completo e responsivo desenvolvido como projeto individual para a disciplina de Desenvolvimento Front-End I. O objetivo do projeto foi aplicar na prática conceitos de estruturação semântica em HTML5 e estilização avançada de layouts utilizando CSS3.

## Tecnologias Utilizadas

O projeto foi construído do zero utilizando as seguintes tecnologias:
- HTML5: Estruturação semântica de todas as páginas (uso de tags como header, main, section, footer).
- CSS3: Estilização visual completa, paleta de cores moderna e tipografia adaptada.
- Flexbox: Utilizado para o alinhamento de componentes de uma linha, como o cabeçalho de navegação (navbar), a seção Sobre Nós e o layout de duas colunas da página de contato.
- CSS Grid Layout: Aplicado na construção de grades complexas e responsivas, como a vitrine da página de serviços e o mural de fotos estilo mosaico na galeria.

## Estrutura de Páginas

O site conta com 5 páginas funcionais totalmente interligadas pelo menu de navegação:

1. index.html (Home): Página de entrada com banner de impacto, slogan chamativo e apresentação histórica da barbearia.
2. servicos.html (Serviços): Vitrine que exibe os cortes e tratamentos oferecidos. Os itens são organizados em formato de cartões (cards) utilizando uma grade automática em CSS Grid.
3. galeria.html (Galeria de Estilos): Um mural visual moderno que simula portfólios reais de cortes, trazendo efeitos de interação (hover) com zoom e máscaras escurecidas nas fotos.
4. agendamento.html (Formulário): Página contendo um formulário estruturado e estilizado com campos para Nome, Telefone, caixas de seleção de serviços, seletores nativos de Data/Hora e campo de texto de observações.
5. contato.html (Contato): Organiza as informações de canais de atendimento, horários detalhados de funcionamento e uma simulação de mapa de localização.
